### 基本信息
| 名称 | 描述 |
| ------------ | ------------ |
| 接口URI | /api/dict/getTree |
| 请求方式 | GET |
| Content-Type | application/json |

### 表单参数
| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| ------------ | ------------ | ------------ | ------------ |
| root | true | boolean | 否 | 是否包含根节点（默认否） |
| pcodes | A,CN,110000 | string | 是 | 父级编码集（逗号分隔） |
